# AI Customer Support Automation

## Overview

This project automates customer support ticket processing using n8n and Google Gemini AI.

## Features

- Gmail Trigger
- AI Email Classification
- JSON Parsing
- Google Sheets Integration
- Automatic Ticket Creation
- Automatic Email Replies
- Category Routing

## Technologies

- n8n
- Google Gemini AI
- Gmail API
- Google Sheets API
- JavaScript

## Workflow

1. Receive Email
2. Analyze using Gemini AI
3. Extract Ticket Details
4. Validate Data
5. Store in Google Sheets
6. Route by Category
7. Send Reply

## Setup

1. Install n8n
2. Configure Gmail credentials
3. Configure Google Sheets
4. Configure Gemini API
5. Import workflow JSON
6. Run workflow

## Sample Categories

- Technical Support
- Billing
- Sales Inquiry
- Refund
- Bug Report
- Feature Request
- Account Access
- General Support
